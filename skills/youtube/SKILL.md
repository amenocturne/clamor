# YouTube Transcript Fetcher

Fetches transcripts from YouTube videos for further processing.

## Features

- Auto-captions and manual subtitles
- Multiple language support
- Outputs clean text transcript

## Requirements

- Python with `uv` for script execution
- `yt-dlp` for video metadata

## Usage

Provide a YouTube URL and the skill will fetch the transcript.
