# Documentation Context

You are now working on documentation. Apply these rules:

- Keep docs high-level. Code is the source of truth for details.
- CLAUDE.md = AI context (commands, architecture, patterns)
- README.md = human context (setup, overview, getting started)
- Update both when behavior changes.
- Use concrete examples over abstract descriptions.
- Don't document obvious things — focus on what's surprising or non-obvious.
